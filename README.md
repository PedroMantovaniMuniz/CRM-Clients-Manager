## 🚀 Sobre o projeto

O **Contracts CRM** é uma aplicação web desenvolvida para centralizar o gerenciamento de clientes, contratos, templates e assinaturas digitais em um único ambiente.

A proposta é oferecer uma experiência simples, moderna e intuitiva para freelancers organizarem seus processos comerciais e documentais com mais praticidade.

---

## ✨ Funcionalidades

* Cadastro e autenticação de usuários
* Gerenciamento de clientes
* Criação e edição de contratos
* Criação de templates reutilizáveis
* Assinatura digital de contratos
* Geração e visualização de documentos
* Interface moderna e responsiva
* Comunicação segura entre front-end e back-end via HTTPS
* Criptografia completa de dados sensíveis

## 🔐 Segurança

O projeto utiliza boas práticas como:

* Variáveis de ambiente para dados sensíveis
* Autenticação com token JWT
* Proteção de headers com Helmet
* Controle de CORS
* Rate limiting para reduzir abuso de requisições
* Comunicação em produção via HTTPS
